# GitHub Pages Personal Website

This is a simple GitHub Pages-ready personal website.

## Files
- `index.html` — main page content
- `style.css` — styling
- `.nojekyll` — prevents GitHub Pages from processing the site with Jekyll

## How to publish on GitHub Pages
1. Create a GitHub repository named `yourusername.github.io`.
2. Upload all files from this folder to that repository.
3. Commit the files.
4. In GitHub, go to **Settings → Pages**.
5. Set the source to the `main` branch and root folder.
6. Your site will appear at `https://yourusername.github.io/`.

## What to edit first
- Replace the placeholder email and links in the Contact section.
- Update the About, Projects, and Publications sections.
- Change the page title in `index.html` if needed.

## Optional additions
- Add a `cv.pdf` file to the repository and link it from the Contact section.
- Add a profile photo and reference it in `index.html`.
- Create additional pages like `publications.html` or `projects.html` later if you want a larger site.
